GrapheneOS app suggestions

GrapheneOS is an opensource operating system for Pixel phones aimed to be the most secure and private OS for mobile phones.

It has some flaws as it sandboxes everything even Google Services. But nothing is unusable at least for my usage. Only thing I encounder, some notifications might delay like whatsapp.

Here you can find some Tips&Tricks and App suggestions.

Tips & Tricks

  • Use work profile for banking apps, just to add an extra layer of security.
  • If you really have to use google services, use different profile and/or work profile only install these apps there.
  • Try to use minimum number of apps to reduce the surface.
  • Try to always look for FOSS alternatives of the apps you want to use.
  • Always use VPN active. Mullvad and Orbot are nice alternatives.
  • Use quality DNS server like nextdns, or mullvad.
  • KISS is a great launcher app, lightweight and fast. Also makes your phone minimal and reduces screentime.
  • Turn off Wi-Fi automatically is a great option.
  • Disable bluetooth whenever you are not using it.

Apps:

  • Mullvad VPN - great VPN service.
  • Aegis - open souce 2fa with backup.
  • Aurora store - Instead of Google Play.
  • Obtainium - Use it instead of F-droid, you can add apps own repositories.
  • Book Reader - pdf, epub… reader.
  • Simple Mobile Tools Calendar - Etar can be used as well, but I like this one better.
  • Simple Mobile Tools Gallery - Aves Gallery is good, but this one is better.
  • Simple Mobile Tools Music Player - simple music player.
  • DAVx5 - to sync calendar, contacts with your nextcloud server.
  • Nextcloud - with your own server.
  • Here WeGo - instead of maps.
  • Joplin - great note taking app, can be synced with DAVx5.
  • K-9 Mail Thunderbird at android domain.
  • KeepassDX - can be synced with nextcloud
  • LibreOffice Viewer - office viewer.
  • Signal - secure messaging app.
  • Tuta - Alternative can be protonmail.
  • ViMusic - listen youtube music.
  • Geometric Weather - simple weather app.
  • Shelter - To manage work profile.
  • Vanadium is the only browser you need.
